Aegis

AI-Powered Data Breach Analyst — RAG, Hybrid Search & Agentic Tool Use

What I Built

Aegis is an AI assistant that answers specific, quantitative questions about cybersecurity data breaches — grounded in the IBM Cost of a Data Breach Report 2024. The project was structured as an incremental build across six steps, starting from a basic LLM chat interface and ending with a full agentic pipeline capable of multi-step reasoning over a structured knowledge base.

The Problem I Was Investigating

Large language models answer from training data — which is stale, unverifiable, and confidently wrong for domain-specific questions. When asked about 2024 breach costs, an out-of-the-box GPT-4o would cite 2023 figures with full confidence. The core question I explored was: how do you ground an LLM in a specific document, and how much does the quality of that grounding matter?

What I Experimented With

I built the system step by step, treating each layer as an experiment. First I connected a basic chat interface to GPT-4o and documented where it hallucinated. Then I added RAG with vector search — which helped but still missed specific data tables. I improved retrieval with hybrid BM25 + vector search, which fixed keyword-match failures. I then replaced the passive RAG pipeline with active tool calling, letting the model decide when and what to retrieve. The final experiment was the most impactful: replacing the naive PDF loader with a layout-aware ingestion pipeline using Unstructured.io and GPT-4o vision to actually read the charts and tables that previous steps were silently dropping.

Key Finding

The biggest leverage point was not the retrieval strategy or the model — it was ingestion quality. A system with hybrid search and tool calling still gave wrong answers because the chart data was never correctly stored in the first place. Fixing ingestion (so ChromaDB contained real ImageSummary and TableSummary chunks from GPT-4o vision) immediately fixed answers that had been wrong across multiple previous steps. RAG is only as good as what was ingested.
